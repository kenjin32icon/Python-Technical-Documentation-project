Introduction

Python is a high-level, interpreted programming language. It is widely used for web development, scientific computing, data analysis, artificial intelligence, and more. Python is known for its simplicity and readability, making it an excellent language for beginners and experts alike.

History of Python

Python was created in the late 1980s by Guido van Rossum as a hobby project. It was named after the Monty Python comedy group. Python quickly gained popularity and is now one of the most widely used programming languages in the world.

Features of Python

  • Easy to learn and use
  • High-level, interpreted language
  • Dynamic typing
  • Support for multiple programming paradigms
  • Large standard library
  • Open-source software

Getting Started with Python

To start using Python, you will need a development environment, such as IDLE, Jupyter Notebook, or PyCharm. You can then write and run Python code, experiment with different features and functions, and create your own programs.

Variables

In Python, a variable is a name that refers to a value. You can assign values to variables and then use them in your code. The value of a variable can be changed at any time.

Creating Variables

To create a variable in Python, use the equal sign (=) to assign a value to a variable name. For example:

  name = "John"
  age = 30

Variable Naming Conventions

In Python, variable names must start with a letter or an underscore. They can contain letters, numbers, and underscores. It is recommended to use lowercase letters and separate words with underscores for better readability. For example:

  first_name
  age_in_years
Data_Types

In Python, the data type of a variable is determined by the value assigned to it. For example, if you assign a string value to a variable, it becomes a string variable. If you then assign an integer value to the same variable, it becomes an integer variable.

Introduction to Python Types

Python has several built-in data types, including integers, floating-point numbers, strings, lists, and dictionaries. Each data type has its own unique characteristics and uses.

Integer Data Type

The integer data type is used to represent whole numbers. It can be positive, negative, or zero. Examples of integers include 42, -7, and 0.

Floating-Point Data Type

The floating-point data type is used to represent decimal numbers. It can be positive, negative, or zero. Examples of floating-point numbers include 3.14, -0.01, and 0.0.

String Data Type

The string data type is used to represent sequences of characters. Strings are enclosed in quotes, either single or double. Examples of strings include "hello world" and 'python'.

List Data Type

The list data type is used to represent ordered sequences of values. Lists are enclosed in square brackets and values are separated by commas. Examples of lists include [1, 2, 3] and ['cat', 'dog', 'mouse'].

Dictionary Data Type

The dictionary data type is used to represent collections of key-value pairs. Dictionaries are enclosed in curly braces and keys are separated from values by colons. Examples of dictionaries include {'name': 'John', 'age': 30} and {1: 'apple', 2: 'banana'}.
Operators

In Python, operators are special symbols that perform specific operations on one or more operands (values or variables). The value or result of the operation is called the result.

Arithmetic Operators

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Modulus: %
  • Exponentiation: **
  • Floor Division: //

Assignment Operators

  • Equal: =
  • Add and Assign: +=
  • Subtract and Assign: -=
  • Multiply and Assign: *=
  • Divide and Assign: /=
  • Modulus and Assign: %=
  • Exponentiate and Assign: **=
  • Floor Divide and Assign: //=

Comparison Operators

  • Equal: ==
  • Not Equal: !=
  • Less Than: <
  • Less Than or Equal: <=
  • Greater Than: >
  • Greater Than or Equal: >=

Logical Operators

  • And: and
  • Or: or
  • Not: not

Identity Operators

  • Is: is
  • Is Not: is not

Membership Operators

  • In: in
  • Not In: not in
Control_Structures

Python has several control structures to control the flow of execution of a program:

  • if statements
  • for loops
  • while loops

Here's an example of an if statement:

  if x > 0:
      print("x is positive")

Here's an example of a for loop:

  for i in range(5):
      print(i)

Here's an example of a while loop:

  i = 0
  while i < 5:
      print(i)
      i += 1
Functions

In Python, functions are defined using the def keyword. The basic syntax is:

  def function_name(arguments):
      # function body

For example:

  def add(a, b):
      return a + b

You can call the function like this:

  result = add(1, 2)
  print(result) # 3
Modules_and_Packages

To use a module in Python, use the `import` keyword, followed by the module name. For example:

import mymodule

mymodule.greet("John")

Creating Packages

To create a package in Python, create a directory containing your modules, and an __init__.py file. The __init__.py file can be empty, but it is necessary for the directory to be recognized as a package. For example, create a directory called `mypackage` containing the file `mymodule.py` and an __init__.py file.

Using Packages

To use a package in Python, use the `from` keyword, followed by the package name and module name. For example:

from mypackage import mymodule

mymodule.greet("John")
